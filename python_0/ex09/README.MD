# ft_package

Small Python package for exercise ex09.

## Install

```bash
pip install .
```

## Usage

```python
from ft_package.count_in_list import count_in_list

print(count_in_list([1, 2, 2, 3], 2))  # 2
```

## Project Files

- `ft_package/count_in_list.py`: main function
- `pyproject.toml`: package metadata
- `requirements.txt`: dependencies
